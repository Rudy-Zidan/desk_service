## Usage
```
createdb lynks_service_desk_development
rake app:db:migrate
rake app:populate
```
## Installation
Add this line to your application's Gemfile:

```ruby
gem 'lynks_service_desk'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install lynks_service_desk
```
